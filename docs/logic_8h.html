union FunctionArg

Function Argument.

This union allows for a function to accept multiple data types as parameters. Since it is marked as a transparent_union it is always passed as a 32-bit integer in the standard registers. Thus, if a function has the signature void foo(u32) it is valid to call it as foo(FunctionArg) . However, void bar(float) would expect the parameter to be in a floating-point register, thus calling bar(FunctionArg) is not valid.

See also
https://gcc.gnu.org/onlinedocs/gcc-3.3/gcc/Type-Attributes.html
