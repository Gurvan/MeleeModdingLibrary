◆ randUint32()

uint32_t randUint32 ( )

Generate random 32-bit value.

Generate random 32-bit value using a linear-feedback shift register. The internal random value of the game is used as a seed. A second random value is generated and written to the address of the internal random random value. This ensures that no random number is used twice.

Returns
Random 32-bit integer
