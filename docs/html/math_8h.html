◆ atan2()

float atan2 ( float  y,
float  x 
)

Compute arc tangent with two parameters.

Returns the arc tangent of y / x, expressed in radians. To compute the value, the function takes into accout the sign of both arguments in order to determine the quadrant.

Parameters
y- Y-coordinate relative to (0,0)
x- X-coordinate relative to (0,0)
Returns
arc tangent of y / x, in interval [-pi, +pi] radians
See also
http://www.cplusplus.com/reference/cmath/atan2/
http://www.embedded.com/design/other/4216719/Performing-efficient-arctangent-approximation
